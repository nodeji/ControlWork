3. ПРАКТИЧЕСКОЕ ЗАДАНИЕ №2 
Создайте таблицу, содержащую шапку и несколько строк. Колонки таблицы: страна, столица, государственный язык. Для оформления текста таблицы задать шрифты, начертание, цвет, размер при помощи таблицы стилей, которая должна быть создана в отдельном файле. Таблица должна быть адаптивной, открываться и читаться на всех устройствах.
